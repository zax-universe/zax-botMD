🆕 UPDATE TERBARU v2.0
🔧 Perbaikan Sistem

· ✅ Global.js - Sekarang menggunakan getter yang lebih dinamis dan ringan
· ✅ Msg.js - Lebih baik dan efisien, tidak ada error message undefined
· ✅ Database - Fix bug database selalu terhapus saat baileys gagal destructure
· ✅ Save System - SaveDb dan saveGc disatukan, lebih mudah diakses & diperbarui
· ✅ Grupify - Fix error destructing saat metadata error/undefined
· ✅ EventEmitter - Peningkatan performa dengan favorit cmd & new cmd
· ✅ Def Command - Fix error saat update command
· ✅ Connection - Sistem koneksi bot lebih dinamis dan ringan
· ✅ Game System - Fix bug sistem game, penambahan sistem cost/kesempatan merampok
· ✅ OCR System - Fix error ocrs tidak bergantung pada prefix
· ✅ Message Security - Sistem pengaman pesan agar jadibot tidak tabrakan dengan sistem utama

💰 Sistem Baru

· 💸 Sistem Pajak - Pajak per-fitur dihitung dengan uang yang kalian punya (setting menyusul)
· 🖼️ Media Thumbnail - Jika thumbnail kosong bisa diganti dengan media gambar
· 👤 Fungsi AFK - Peningkatan fungsi afk
· 🌾 Auto Farm - Sistem auto farming
· 👥 Jadibot - Sistem jadi bot dengan cmd khusus
